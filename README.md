# Cook's Compass

Cook's Compass is a lightweight, static recipe and cooking companion site. It is built with plain HTML, CSS, and JavaScript to keep the experience fast, easy to host, and simple to customize. The project focuses on a clean layout, approachable content, and a small asset footprint so it can be deployed anywhere without a build step.

## Project Goals

- Provide a welcoming front door for home cooks with a clear, navigable layout.
- Keep the codebase easy to understand and modify without frameworks.
- Use static assets for reliability and low hosting cost.
- Offer room to grow into new pages or interactive features.

## Key Features

- Single-page layout with structured sections.
- Custom styling in a dedicated CSS file for fast iteration.
- Lightweight JavaScript for any interactive behavior.
- Favicons and a web manifest for modern browser support.

## Tech Stack

- HTML for structure
- CSS for styling
- JavaScript for interaction

## Folder Structure

- [index.html](index.html): Main page markup.
- [styles/main.css](styles/main.css): Site styles.
- [scripts/main.js](scripts/main.js): Site behavior.
- [images/](images/): Icons, manifests, and static assets.

## Getting Started

1. Open [index.html](index.html) in a browser.
2. Edit [styles/main.css](styles/main.css) to change the look and feel.
3. Update [scripts/main.js](scripts/main.js) to add or change behaviors.
